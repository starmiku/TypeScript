# TypeScript基础语法

TypeScript程序由以下几个部分组成：

* 模块
* 函数
* 变量
* 语句和表达式
* 注释

## TypeScript程序

```ts
const hello : string = "Hello World!"
console.log(hello)
```

首先需要进行编译得到以下js代码

```js
var hello = "Hello World!";
console.log(hello);
```

## TypeScript格式

TypeScript会忽略程序中出现的空格、制表符和换行符。空格、制表符通常用于缩进代码。

TypeScript区分大小写。

分号在TypeScript中是可选的，建议使用，如果语句写在同一行则一定需要使用分号来分隔，否则会报错。

## 循环语句

TypeScript支持以下几种循环语句：

* for循环
* while循环
* do while循环
* for in循环
* for of循环
* forEach循环
* every循环
* some循环

流程控制：

* break语句
* continue语句

### for in循环

for...in语句用于遍历对象的可枚举属性/键名/索引。

```ts
let person = { name: "John", age: 30, city: "New York" };

for (let key in person) {
    console.log(key); // 输出 "name", "age", "city"
}
```

### for of循环

for...of语句用于遍历可迭代对象（如数组、字符串、Map、Set等）的键值。

```ts
let arr = [1, 2, 3, 4, 5];

for (let value of arr) {
    console.log(value); // 输出 1, 2, 3, 4, 5
}
```

如果对象不可迭代（如对象字面量），则会抛出错误。此时可使用Object.keys()和Object.values()方法来获取对象的键和值。

```ts
let person = { name: "John", age: 30, city: "New York" };

for (let key of Object.keys(person)) {
    console.log(key); // 输出 "name", "age", "city"
}

for (let value of Object.values(person)) {
    console.log(value); // 输出 "John", 30, "New York"
}
```

### forEach循环

forEach()方法对数组的每个元素执行一次回调，没有返回值（undefined）。相当于用 for 循环做操作，但更简洁。

```ts
let arr = [1, 2, 3, 4, 5];

arr.forEach(function(value, index) {
    console.log(value, index); // 输出 1 0, 2 1, 3 2, 4 3, 5 4
});
```

forEach() 的回调函数最多可以接收3个参数（value、index、array），但最少只需要1个。你可以根据需要只传需要的参数，多余的忽略即可。

且 forEach() 的每次回调都会创建一个新的函数作用域，因此不会出现 var 循环变量的作用域问题。

### every循环

every()方法用于检测数组中的所有元素是否都满足指定条件（即回调函数返回true）。它返回一个布尔值。当遇到第一个不满足条件的元素时，停止遍历，并返回false。

```ts
let arr = [2, 4, 5, 8, 10];

arr.every(value => {
    console.log(`当前值：${value}`);
    return value % 2 === 0;
});
// 输出：
// 当前值：2
// 当前值：4
// 当前值：5
// false
```

### some循环

some()方法用于检测数组中的是否有至少一个元素满足指定条件（即回调函数返回true）。它返回一个布尔值。当遇到第一个满足条件的元素时，停止遍历，并返回true。

```ts
let arr = [1, 4, 5, 8, 10];

arr.some(value => {
    console.log(`当前值：${value}`);
    return value % 2 === 0;
});
// 输出：
// 当前值：1
// 当前值：4
// true
```
