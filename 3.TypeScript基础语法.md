# TypeScript基础语法

TypeScript程序由以下几个部分组成：

* 模块
* 函数
* 变量
* 语句和表达式
* 注释

## TypeScript程序

```ts
const hello : string = "Hello World!"
console.log(hello)
```

首先需要进行编译得到以下js代码

```css
var hello = "Hello World!";
console.log(hello);
```

[TypeScript保留关键字](https://www.runoob.com/typescript/ts-basic-syntax.html)

## TypeScript格式

TypeScript会忽略程序中出现的空格、制表符和换行符。空格、制表符通常用于缩进代码。

TypeScript区分大小写

分号在TypeScript中是可选的，建议使用，如果语句写在同一行则一定需要使用分号来分隔，否则会报错。
